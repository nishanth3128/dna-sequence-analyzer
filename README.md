DNA Sequence Analyzer:

A beginner-friendly bioinformatics project built using Python that analyzes DNA sequences and performs basic genomic comparisons.

Features:
-> DNA sequence validation
-> ATGC base counting
-> GC percentage calculation
-> Mutation detection between sequences
-> Sequence similarity comparison
-> Interactive command-line interface

Concepts Used:
-> Python loops
-> Conditional statements
-> String processing
-> Input validation
-> Basic bioinformatics logic

Example Usage:
Enter your DNA sequence: ATGCGTA

A Count: 2
T Count: 2
G Count: 2
C Count: 1
GC%: 42.86%

Mutation at position 3: G → A
Similarity: 85%

How to Run
1.Install Python (3.x)
2.Clone repository
git clone https://github.com/nishanth3128/dna-sequence-analyzer
